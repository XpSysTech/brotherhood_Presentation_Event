## Overview

This repository contains the source code for a **single-page informational website** created for a **one-day, in-person investment information session** hosted by the Boena Investment Fund.

The page was intentionally designed as a **temporary, event-specific site** to provide clarity, structure, and professionalism for attendees before and during the session. It is not intended to function as a permanent public website.

The visual and structural style is inspired by **large government, institutional, and private-sector forums and conferences**, prioritizing clarity, hierarchy, and restrained presentation over branding or marketing.

---

## Purpose

The primary objectives of this page are to:

- Present a clear overview of the Boena Investment Fund initiative
- Communicate the meeting agenda and expectations
- Provide structured responses to common member questions
- Offer clear contact information for follow-up and virtual attendance
- Maintain a professional, neutral, and informational tone

Attendance at the event is **informational only** and does not imply commitment.

---

## Design Philosophy

- **Single-use, event-focused**: Built specifically for one session
- **Institutional tone**: Inspired by government briefings and investment forums
- **Clarity over persuasion**: Informational, not promotional
- **Structured layout**: Clear sections, predictable navigation
- **Minimal interactivity**: No JavaScript or external dependencies
- **Accessible presentation**: Readable typography and spacing

---

## Technical Notes

- **Single HTML file**
- Embedded CSS (no build step required)
- Responsive layout (desktop and mobile)
- Sticky navigation for ease of reference during scrolling
- No backend, tracking, or analytics
- No form submissions or data collection

The page can be opened locally or hosted on any static file server.

---

## Scope & Limitations

This page:

- Is not a legal or financial prospectus
- Does not process payments or collect personal data
- Does not represent a permanent digital presence
- Is not intended for ongoing marketing or recruitment

Future sessions or initiatives may require separate materials or platforms.

---

## Usage

The site may be:

- Shared via direct link prior to the event
- Displayed on devices during the session
- Archived after the event for reference or documentation

Modifications for future events should be done by duplication rather than extension.
